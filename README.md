# Quick Bite 🍔

Quick Bite is a responsive food ordering web app built with HTML, CSS, and JavaScript.  
It focuses on real user interactions and UI patterns commonly found in e-commerce apps.

## 🚀 Live Demo
https://quickbbite.netlify.app/

## 📸 Features

- 🛒 Dynamic cart with quantity controls
- 💾 Persistent cart using localStorage
- 📱 Responsive layout (mobile + desktop)
- 🧾 Checkout modal with form
- 🔔 Toast notifications for feedback
- 📦 Cart drawer (toggle + overlay + close button)
- ➕ Add to cart interactions with real-time updates

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox, responsive design)
- Vanilla JavaScript (DOM manipulation, events, state management)
- LocalStorage (data persistence)

## 🧠 What I focused on

This project was built to practice:

- Managing UI state without frameworks
- Structuring interactive components (cart, modal, toast)
- Handling user events and feedback
- Building reusable and scalable frontend logic
- Creating a clean and responsive UI
